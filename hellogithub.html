С помощью Git программисты и разработчики ориентируются в коде отслеживают измененияю Git помогает вернуть файлы в исходное состояние и видеть изменения, внесённые в определённый период. Разработчик выполняе разные команды(например, commit, push), а всеизменения синхронизируются с центральным репозиториемю Git - это система контроля версий, а GitHub - онлайнсервис, по сути социальная сеть. Одна из основныч целей GitHub- быть единым местом для проекта с исходным кодомю Предполагается, что пользователь делится чум-то полезным, а другие люди смогут участвовать в разработке.
